# Sobre o projeto:

### Um amigo que tem uma loja de roupas precisava cadastras uma lista de contatos em um boot para whatsapp. Porém sua lista de contatos estava cadastrada de uma forma diferente do que o programa precisava. 

(81)91234-5678 estava dessa forma.

5581912345678 precisava estar assim.

### Para não ter que fazer essa mudança manualmente em um excel de mais de 3.000 contatos. Fiz essa automação para ele utilizando python.

### Linguagem principal: 
Python
### Bibliotecas usadas: 
Pandas - para manipular os arquivos excel.
Tkinter - para fazer a interface do usuário 
### Ferramentas: 
PyInstaller - para gerar o executável
